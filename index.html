🔗 GMD 바로가기란?

자주 사용하는 사내 시스템과 문서에 빠르게 접근할 수 있는 바로가기 모음입니다.

✏️ 편집 모드

  • 편집 모드 - 상단 왼쪽 버튼을 클릭하여 활성화
  • 바로가기 수정 - 편집 모드에서 ✏️ 버튼 클릭
  • 바로가기 추가 - 편집 모드에서 ➕ 버튼 클릭
  • 바로가기 삭제 - 편집 모달에서 삭제 버튼 클릭
  • 데이터 초기화 - 설정에서 모든 편집 내용 초기화

⚙️ 설정 옵션

  • 다크 모드 - 다크/라이트 테마 전환
  • 새 창에서 열기 - 바로가기를 새 탭에서 열지 현재 탭에서 열지 선택
  • 간략하게 표시 - 설명을 숨기고 컴팩트하게 표시

📁 카테고리 관리

  • 접기/펼치기 - 카테고리 헤더를 클릭하여 접거나 펼칩니다
  • 순서 변경 - 설정에서 카테고리 순서를 변경할 수 있습니다

💡 툴팁

바로가기 위에 마우스를 올리면 상세 설명을 볼 수 있습니다.

🏷️ 아이콘 설명

  • Confluence 문서 링크
